### 收费

1、github-copilot  -  这两年呼声少了，我得为它正下名，效果提高了很多，支持常用IDE，最新模型，小功能实现很强， 每月花费70

2、cursor  - 今年各种渠道下被claude code的消息盖过，但是我测试某些情况下是超过 claude code -应该是内置了有效提示词，总体依旧好用，就是贵，每月基础版花费140

3、claude code - 有了中转渠道开始使用它，能力不错（可能中转原因没有体现出强能力），集成到了Obsidian，随时开终端解答问题，提生很多效率 ，可以使用各种skill 设计软件也不用了，每月中转花费50

4、open code - 主要支持国内模型API对接，同时也支持github-copilot 、价格、之前使用GML4.7 效果一般 ，最近新默认能力上来了，效果对比claude code一点不差， 最近阿里云 codingplan 价格给力 ，每月花费 40

### 免费

1、豆包  手机端对话
2、deepseek、chatgpt  gemini 这三个常用的问题和解决方案，各有混用，gemini 回复时效性、客观性上比deepseek要高些，另外就是gemini 生成图片好用的很。
